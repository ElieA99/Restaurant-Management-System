How to check the .Net Framework Version Installed on Your PC ?

Firstly Run CMD as Administrator

As Seen in the Image

  1. Search for CMD in the search bar
  2. Select it
  3. Then Finally Run it as Administrator
2nd step we will be using the followin command

Just Copy it from Below and paste it in the CMD

reg query "HKLM\SOFTWARE\Microsoft\Net Framework Setup\NDP" /s

Finally

And here you go !! now you can see the version as example mine is 4.8.04084
