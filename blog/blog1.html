Front End Form Invoice/Order Input

Front End Form: Invoice/Order Input

This is the first form that is shown after the Application is launched.

Note: Phone-Number,Names,Prices are only for tetsting and are not related to anyside


The Tool-Strip Bar contains:
  1. Admin-DashBoard: This button show the Login form that connects the Front-End with the Admin-DashBoard.
  2. Guide web Page: Redirects the user to the guide page on the website(for sure internet is required in this case).
  3. Delivery-Data: In this form you Insert customers Home address in case he wanted his order to be delivered.
  4. Export to excel: This Feature let's you export the input order in the datagridview as excel format.
  5. About: Shows you everything related to the App Version/Copy Right/Developper Name.
  6. Exit: Close The Application Completley.

Order-Input

A Example On a Order Input, simply choose from the Group Box the product you need with the Quantity and press the Add button. After Your Done ,select from the options section if the order is Dine-In or Take Away Or Delivered with the payment method then Click On the sub-Total Button, if u need to give the customer a disscount you can else leave it 0,in the end press the Total Button to the the final amount. In case the order needed to be delivered then the Delivery-Charge Section will be enabled then you can type the charge if there is no charge keep it 0.

Clear Button lets you clear the datagridview and generates a new invoice id after each order-input press it.
The Delete unwated row option let you delete a row from the datagridview by selecting it (in case you inserted by mistake a unwated product)
