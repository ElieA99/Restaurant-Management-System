Admin Dashboard

Backend Form: Admin Dashboard

This Back-End Form is Accessed by the Owner Of The Restaurant,Its has many features and lets him Generate Reports,
It Contains many section.

Starting With the Settings Section
  1. Employees
  2. Configurations
  3. Login-Configuration
  4. Expenses
  5. Orders-History
  6. Net-Profit

1-Employees:
This Form lets you ADD And manage your Employees with The Monthly Sallary Given.

2-Configurations:
This Form lets you Configure To ADD Resto-Name/Phone Number/Region and The Currencu Used To The Invoice.

3-Login-Configuration:
This Form let's You ADD/DELETE Login Credentials As You Need.

4-Expenses:
In This Form You Can Manage You Expenses To Track Them.As Example Each Time You Make A Payment You directly ADD it to the Database and how mush it cost you,You can generate a Report in the Reports section.

5-Orders-History:
Each Time you have an Printed Order The Neccessary Data are stored in the specified Databse so you can track the Orders-Profit.You can generate a Report in the Reports section.

6-Net-Profit:
This Form gives you with a click of button all your Expenses with the Orders-Profit and the Net-Profit Amount,You can generate a Report in the Reports section.

Beverages Section

This Section is divided into 5 parts each part has the same feature to Add Product,Update Its Price and to Delete it in case its not available anymore.

Menu Section

This Section is divided into 8 parts each part has the same feature to Add Product,Update Its Price and to Delete it in case its not available anymore.

Reports Section

In this Section you can Generate detailed Report of the 3 main sections

  1. Orders-History
  2. Expenses
  3. Net-Profit

Exit/Logout Buttons

The Exit Button (Alt+E) Close The Application Completley.

And The Logout Button (Alt+L) Redirects the Admin from the Dashboard to the First Form.
